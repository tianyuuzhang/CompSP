# 经验 ASR 分辨率限制

当前 txfilled 数据通常每条指令 `totQA=20`，因此经验 ASR 的最小非零步长是 `0.05`。
这意味着 `0.01` 与 `0.001` 不能由单条指令的 20 次经验 ASR 直接区分，只能作为模型预测风险或序贯增采目标。

## 总体

| 指标 | 数值 |
|---|---:|
| n | 88683 |
| unique_asr_count | 21 |
| min_nonzero_step | 0.05 |
| zero_asr_fraction | 0.1690515656890272 |
| low_asr_le_0_05_fraction | 0.20425560704982917 |
| low_asr_le_0_10_fraction | 0.2300666418592064 |
| median_ci_width | 0.20536960366783857 |
| zero_asr_95_upper_mean | 0.11663898290487536 |
