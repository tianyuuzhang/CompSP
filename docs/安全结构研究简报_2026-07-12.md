# 安全结构研究简报

日期：2026-07-12，最后维护：2026-07-14

## 一句话结论

当前最稳的结论是：少量模型回答可以作为黑盒风险嗅探信号，帮助定位更高 ASR 的指令；但它主要体现行为风险结构，不应过早等同于白盒单一安全方向。

2026-07-14 口径补充：ASR 是优先利用目标，ALR 和替代安全定义主要用于探索和稳健性检查；研究对象不局限于 Llama，也不局限于长回答内容。黑盒侧允许使用 q1、回答文本、拒绝形式、长度、格式、unsafe 计数和其他文本统计特征；白盒侧可以同时研究输入端与输出端结构，用来解释黑盒信号的来源。

## 今日新增

- 新增 `scripts/simulate_sequential_asr_sampling.py`，用于离线模拟少量回答后的序贯 ASR 增采。
- Llama 三攻击首轮结果：随机 top 20% 的后续 ASR 约 `0.636`；使用前缀回答选择候选后，`k=1/2/4` 的 hazard-weighted 策略分别约 `0.770/0.802/0.824`。
- 主要盲点：若前缀回答没有 unsafe，经验 ASR 类策略几乎无法召回后缀高风险样本；ALR 对此更敏感，但总体风险提升弱于 ASR/hazard 策略。
- prefix-zero 二阶段模型新增 TF-IDF 高权重词项审计。文本信号确实能在“前几次全安全”的子集中检出后续高风险样本，但高权重项混有主题词、模板残片和拒绝结构，下一步需要做同题/同攻击方案内的主题控制。
- prefix-zero 三视图对照完成。仅回答文本稳定强于仅 `q1`：`k=1/2/4` 的 AP 分别为 `0.414/0.195/0.141`，而 `q1` 只有 `0.216/0.061/0.009`；`k=4` 的 top recall 从 `0.062` 提升到 `0.812`。这支持回答内容提供输入攻击文本之外的黑盒嗅探信号。
- 多 seed 和分攻击方案复验完成。三攻击混合 5 seed 下，response AP 为 `0.457/0.211/0.069`，q1 AP 为 `0.230/0.057/0.012`；OFA、PAIR、DrAttack 的 `response-q1 AP` 均为正，说明该增量不是单一攻击方案造成。
- 回答归属打乱负对照完成。打乱后 response AP 降为 `0.268/0.100/0.034`，top recall 降为 `0.377/0.352/0.282`，说明具体回答归属确实贡献了预测力；但打乱后仍高于随机，说明题目/模板边际分布仍有残留信号。
- 强词面掩码负对照完成。掩掉拒绝词、危险词、模板词、格式词和诱饵词后，response AP 为 `0.465/0.220/0.071`，几乎不低于原始 `0.457/0.211/0.069`。这说明当前 response 增量不能被显式词面捷径完全解释。
- 低维 SVD 语义探针完成。强掩码 response 的 32 维 SVD Ridge AP 为 `0.436/0.187/0.053`，接近全 TF-IDF；最佳单分量 AP 为 `0.378/0.231/0.083`。这说明剩余文本信号具有低维语义/姿态结构，但还不能等同于内部安全方向。
- 固定预算追加采样模拟完成。hazard 策略 selected future ASR 最高，但 prefix-zero 高风险召回几乎为 0；成本归一化的 `extra/row=4` 下，混合策略 `hybrid_hazard_text_q0.5` 在 `k=1/2/4` 的 prefix-zero 高风险召回达到 `0.611/0.765/0.724`，final recall 达到 `0.860/0.917/0.951`，均高于 hazard 和随机。
- 成分受控信息增益实验完成。固定前缀成分后，`response_only` 仍强于 count baseline 和回答归属打乱：`safe3_unsafe1` 中所有样本前缀 ASR 都是 `0.25`，但 response Spearman 为 `0.346`，count baseline 为 `0.133`，打乱回答为 `0.198`。这说明少量回答文本的预测力不只是来自前缀 unsafe 比例。
- 成分受控实验已补 `future_alr` 和三攻击分层。ASR 增量主要来自 OFA/PAIR：`safe3_unsafe1` 中 OFA response Spearman 为 `0.584`，PAIR 为 `0.295`；DrAttack 为 `-0.056`，不支持同样结论。ALR 中 count baseline 已很强，说明 ALR 更多反映长度/格式和拒绝姿态，不宜作为“具体长内容语义增量”的主证据。
- 自然前缀条件增量实验完成，并按“80 问题重抽 20 测试题”做两个全局重随机复验。控制 `count+q1` 后加入 response 的 ASR 增量很小但可复验：random80 split `20260714/20260715` 在 `k=1` 的 Spearman delta 分别为 `0.032/0.040`，`k=2` 为 `0.017/0.028`。该结果支持“弱黑盒行为观测增量”，但 shuffled 接近 full，不能升级为强机制证据。
- 自然前缀条件置换完成。两个 random80 split 中，response 的 Spearman 和问题宏平均 pairwise 增量均显著高于同数据集、同问题、同自然组成桶置换的 null；AP 增量较小，`k=4` 不稳定。该结果可以作为等级 0 行为预测分支的收束证据，但仍不是内部 H 暴露证据。
- H_proxy 管线 smoke 完成。使用 legacy 20 题 PAIR/DrAttack 的 `wb_last/wb_mean` 逐层拒绝方向投影时，`count+q1` 已能较强预测 proxy，response 只有小幅正增量。该结果只说明等级 1 管线可运行，不能作为内部结构暴露结论。
- 当前进度检查：远端无本项目长实验进程，GPU 空闲；已有结果已经落档到周报和本简报。下一步若启动白盒 `H_input/H_start` 缓存，先跑 `10~20` 分钟 smoke，再按 tqdm 估计全量耗时，初步预计三攻击 0-79 问题约 `2~5` 小时。

## 当前长实验

ASR 伪安全方向三攻击训练已经完成。OFA accuracy/AUC 为 `0.6103/0.6580`，PAIR 为
`0.6485/0.7086`，DrAttack 为 `0.5656/0.5923`。DrAttack 高于随机但明显弱于
OFA/PAIR，说明 ASR 伪方向可学习性存在攻击方案差异，不能把三者混成同一强结论。

## 下一步

- 对 ASR-DrAttack 做稳定性或 bootstrap 复验，并和 ALR-DrAttack 的近随机结果对照。
- 扩展序贯增采到跨模型 OFA 和 HarmBench。
- 把文本残差嗅探器加入增采策略，重点寻找“前缀安全但后续高风险”的隐蔽样本。
- 新增的 prefix-zero 保留名额实验显示，隐蔽高风险召回可以提高到 `0.38~0.69`，但平均后续 ASR 会下降；后续需要把“攻击优化”和“安全审计”两类预算目标分开报告。
- prefix-zero 二阶段模型进一步显示：在前 `k=4` 次均无 unsafe 的样本中，高风险比例只有 `1.0%`，但 `text_ridge` 的 top 20% recall 达到 `0.812`，说明回答具体内容可用于发现低频隐蔽风险。
- 主线下一步：补全 0-79 问题的白盒 `H_input/H_start` 缓存，定义 harmfulness、refusal/compliance 和 generation trajectory 多维坐标，再检验 response 是否在 `count+q1` 之后提高内部状态预测；topic-matched control 和新回答采样作为行为分支的补充。
