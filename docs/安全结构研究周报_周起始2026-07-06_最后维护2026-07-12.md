# 安全结构研究周报

周起始：2026-07-06  
最后维护：2026-07-13

## 本周主线

本周研究从“CompSP 是否学习到单一安全向量”扩展为更宽的三方关联问题：指令攻击性、白盒内部安全结构、黑盒可观测回答内容之间是否存在稳定关系。当前口径是：ASR 优先作为结构利用目标，ALR、替代 ASR、拒绝形式和白盒伪方向作为探索变量；黑盒实验只使用攻击者可见文本和文本统计特征。

## 已完成结果

- ASR 伪安全方向已经完成 OFA、PAIR 两组训练，DrAttack 仍在运行。已完成结果中，OFA 测试准确率约 `0.610`，PAIR 测试准确率约 `0.648`。
- ASR 伪安全方向三攻击已在 2026-07-13 完成：OFA accuracy/AUC 为 `0.6103/0.6580`，PAIR 为 `0.6485/0.7086`，DrAttack 为 `0.5656/0.5923`。DrAttack 是弱正结果，明显低于 OFA/PAIR。
- 回答内容分析显示，少量回答文本能预测后续风险。严格前缀/后缀拆分下，单条回答 TF-IDF 对后续 ASR 的 Spearman 约 `0.737`，四条回答约 `0.788`。
- 跨模型 OFA 结果显示，回答文本相对 q1-only 有稳定增量；去除显式拒绝词和危险词后，ASR 平均仍比 q1-only 高约 `0.069`。
- 残差分析表明，ASR 和 hazard-weighted ASR 在扣除长度、拒绝和格式后仍有弱但可测的文本增量；ALR 大部分由长度、拒绝和格式解释。
- HarmBench 复验支持“存在文本增量”，但目标模型依赖更强。该数据集每个目标模型测试题较少，DeepSeek 等模型标签方差不足，不能过度泛化。
- 经验 ASR 分辨率分析确认，`totQA=20` 的经验 ASR 最小步长为 `0.05`，不能直接区分真实 ASR `0.01` 与 `0.001`。
- 新增序贯 ASR 增采模拟：在 Llama 三攻击数据中，只看前 `k=1,2,4` 条回答选择 top 20% 候选，后续 ASR 从随机约 `0.636` 提升到最高约 `0.824`。
- 序贯增采已扩展到 JBB-OFA 与 HarmBench-OFA 的 Llama、Qwen、4omini、DeepSeek。`k=4` 的 hazard-weighted 策略在所有目标模型上均高于随机；例如 JBB-Qwen 从 `0.386` 升到 `0.644`，HB-Qwen 从 `0.570` 升到 `0.750`。
- prefix-zero 隐蔽风险模型显示，若前 `k` 次回答均未出现 unsafe，回答文本仍能预测后缀高风险。三视图对照进一步确认：仅回答文本明显强于仅 q1，`k=4` 的 top recall 为 `0.812`，而 q1-only 为 `0.062`。这使“回答内容提供输入之外的黑盒嗅探信号”成为当前较强的阶段性结论。
- 5 seed 复验后结论仍成立：三攻击混合下 response AP 为 `0.457/0.211/0.069`，q1-only 为 `0.230/0.057/0.012`；按 OFA、PAIR、DrAttack 分层后 response-q1 AP 均为正。
- 回答归属打乱负对照显示，打乱同题同攻击内的 response 后，AP 降到 `0.268/0.100/0.034`，top recall 降到 `0.377/0.352/0.282`。这支持“具体回答内容与后续风险有对应关系”，但打乱结果仍高于随机，提示边际分布和回答姿态残余仍需控制。
- 强词面掩码负对照显示，掩掉拒绝词、危险词、模板词、格式词和诱饵词后，response AP 仍为 `0.465/0.220/0.071`，top recall 为 `0.568/0.605/0.700`。这说明剩余信号不只是显式词面捷径，更可能包含回答姿态、主题语义或更深层的行为结构信息。
- 低维 SVD 语义探针显示，强掩码 response 的 32 维分量可保留大部分预测能力，SVD Ridge AP 为 `0.436/0.187/0.053`；单个最佳分量也有明显信号。这支持“回答文本剩余信号具有低维可学习结构”，但分量仍混合拒绝姿态和具体主题，不能直接等同于白盒安全方向。
- 固定预算追加采样模拟显示，prefix-zero 嗅探器在安全审计目标上有实际价值。成本归一化的 `extra/row=4` 下，`hybrid_hazard_text_q0.5` 的 prefix-zero 高风险召回为 `0.611/0.765/0.724`，显著高于 hazard 的 `0.006/0.007/0.000`；final recall 为 `0.860/0.917/0.951`，均高于随机和 hazard。但 selected future ASR 低于 hazard，说明攻击优化和安全审计是两个不同目标。
- 成分受控实验进一步排除了前缀 `asr_small/alr_small` 的朴素解释。固定 `3` 条安全 + `1` 条危险时，所有样本前缀 ASR 都是 `0.25`；`response_only` 预测后缀 ASR 的 Spearman 为 `0.346`，高于 count baseline 的 `0.133` 和回答归属打乱的 `0.198`。危险回答-only、安全长回答-only 口径下也有同方向结果。
- 成分受控实验补充 `future_alr` 与按攻击方案分层后，结论更细：ASR 的 response 增量主要来自 OFA 和 PAIR，DrAttack 在 `safe3_unsafe1` 严格口径下不成立；ALR 中 count baseline 已很强，说明它主要由长度/格式和拒绝姿态解释。
- 自然前缀条件增量实验完成并做两个全局重随机复验。该实验不事后挑选回答组成，而是在 `count+q1` 基础上加入 response。random80 split `20260714/20260715` 中，`k=1` 的 Spearman delta 为 `0.032/0.040`，`k=2` 为 `0.017/0.028`，`k=4` 为 `0.008/0.017`。这支持一个弱但可复验的 response 增量，同时说明效应远小于前缀计数和 q1 本身。
- 条件置换检验进一步确认，自然前缀 response 增量高于同数据集、同问题、同自然组成桶置换 null；Spearman 和问题宏平均 pairwise 在两个 random80 split 中稳定显著，AP 增量很小且 `k=4` 不稳定。该结论仍属于行为预测层，不能写成内部安全结构暴露。

## 重要解释修正

- 当前证据更强地支持“模型输出文本携带行为风险信息”，而不是直接证明“输出文本线性读取了白盒单一安全方向”。
- “长回答更有信息”不是稳定结论。长度匹配后，伪方向信号明显减弱；ASR/ALR 仍保留一部分信号，但强度下降。
- 少量回答增采策略对已经出现 unsafe 迹象的指令很有效，但对“前缀零 unsafe、后缀高风险”的隐蔽样本召回很低。后续需要结合 ALR、回答形态和文本残差模型。
- 跨模型序贯增采不能被解释为“使用了跨模型标签”：每次选择都在同一目标模型、同一问题内完成，汇总表只是事后比较不同目标模型的效果。

## 下周优先任务

- 对 ASR-DrAttack 做稳定性或 bootstrap 复验，并与 ALR 三攻击结果并列表格。
- 将序贯增采从“前缀 unsafe 证据”升级为“前缀文本残差嗅探器”，优先提高前缀零 unsafe 但后续高风险样本的召回。
- 把自然前缀框架的 target 从 future ASR 改为白盒 `H_input/H_start` 或多维安全子空间坐标；topic-matched control、新回答采样和 DrAttack 诊断作为行为分支补充。
- 用 frozen sentence embedding 和目标模型 hidden state 复验 SVD 结论，并检查回答文本分量是否与白盒伪方向或多维安全子空间一致。
- 针对低 ASR 精细定位，考虑跨同题 prompt 族的层级模型，而不是逐指令独立估计。
- 若要加强机制证据，继续做多维白盒结构，而不是把单一伪方向作为唯一内部安全结构。
