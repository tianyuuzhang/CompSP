# 文档

这里放的是清理后项目的说明、实验索引和溯源笔记。

- `experiment_map.md`：论文实验到代码与数据的对应关系。
- `source_audit.md`：旧脚本哪些有效、哪些已被替代、哪些只保留索引。
- `data_index.md`：原始/整理后/分析用数据的位置索引。
- `model_index.md`：实验会用到的基座模型和 LoRA 路径索引。
- `asr_data.md`：`txfilled` 型 ASR/ALR 数据集的统一接口与校验说明。
- `asr_validation_summary.md`：当前注册数据的校验汇总。
- `safety_linear_structure_research_plan.md`：把 CompSP 误差与白盒安全方向联系起来的研究计划。
- `safety_linear_paper_notes.md`：已下载安全线性方向相关论文的中文笔记。
- `whitebox_compsp_error_alignment_experiment.md`：白盒方向与 CompSP 对齐实验的具体方案。
- `whitebox_llama_alignment_first_run.md`：第一次 Llama 白盒方向实验的初跑记录。
- `代码与实验逻辑审计.md`：本次全面核查的结论、修复项和验证记录。
- `待确认问题.md`：需要用户后续确认的研究口径、数据缺口和解释风险。
- `长期研究计划.md`：前期 CompSP 工程收口与后续安全结构研究的长期计划。
- `安全结构研究总计划.md`：当前安全结构方向的核心问题、假设分层、实验路线和验收标准。
- `安全结构研究TODO.md`：后续实验、工程、文献和写作任务清单。
- `安全结构研究讨论.md`：用于持续和用户讨论假设、反例、实验结果与决策的问题记录。
- `pseudo_safety_learnability.md`：伪安全结构可学习性实验的设计、代码入口和结论边界。
